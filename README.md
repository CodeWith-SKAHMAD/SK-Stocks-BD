# 📈 BD Stock Tracker

বাংলাদেশ স্টক ইনভেস্টমেন্ট ট্র্যাকার — DSE ও CSE উভয় মার্কেটের জন্য।

## ফিচার
- ✅ পোর্টফোলিও ড্যাশবোর্ড (লাভ/ক্ষতি, সাপ্তাহিক/মাসিক)
- ✅ BD Market open/close সময়
- ✅ Buy/Sell ট্রানজেকশন
- ✅ TradingView চার্ট
- ✅ Watchlist
- ✅ Fibonacci সিগন্যাল
- ✅ Calculator (লাভ/ক্ষতি, Break-Even)
- ✅ Light/Dark Mode
- ✅ Mobile Responsive

## Tech Stack
- React + Vite
- Supabase (Auth + Database)
- GitHub Pages (Hosting)
